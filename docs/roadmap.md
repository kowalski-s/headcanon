- [x] **Writer editor (W1).** TipTap, главы, автосейв, «Мои истории», publish toggle. Сделано 2026-05-27; рестайл в этапе A.

## Phase 2 — Generation + Audio

**Цель:** secondary-путь и аудио-USP.

- Generator path (уже построен в v1) репозиционируется как secondary entry; AI-generated в отдельной категории, видны по умолчанию.
- **Аудио** — TTS-прослушивание для обоих типов историй (writer-written и AI-generated). Underserved формат, late-night headphones-on. Включено в PRO. Несколько голосов, скачивание как подкаст-эпизод.

## Phase 3 — Video (original worlds only)

**Цель:** видео-differentiator в безопасной юридической зоне.

- Только `storyHasCanonicalIp === false` (оригинальные миры). Tag-gating из Phase 1.
- Moments 5-15 сек, не полные эпизоды. Анимированные иллюстрации (motion stills, parallax) как дешёвая альтернатива.
- Замороженный watch-режим из v1 размораживается и переориентируется. Async-очереди с push.
- Полное видео — через 12-18 мес, когда цены упадут до $5-8/эпизод, на WHALE tier.

## Phase 4 — Competitive + virality

**Цель:** engagement loop, удержание.

- **Соревновательная механика.** Авторы в топах получают бесплатные генерации, иллюстрации, месяц PRO/STUDIO, бейджи, featured.
- **Метрика топа: сохранения + дочитывания** (не просмотры/лайки — накручиваются). Engagement-комментарии. Трекинг-события закладываются в Phase 1.
- Категории топов: по фандомам, по типам историй, по объёму, по времени, отдельный топ оригинальных миров (мотивация писать своё).
- Виральное: «что если» (branching), remix, reader cameos, cross-fandom mashups для TikTok.

## Phase 5 — Monetization

**Цель:** можно платить. Tier-структура — [tech-stack.md](tech-stack.md#tier-структура-phase-5-закладывается-заранее).

- YooKassa, RU-payments, подписки.
- FREE / PRO (199–299 ₽) / STUDIO (490–690 ₽) / WHALE (по запросу) — рублёвые тиры от якорей ниши (Фикбук-премиум 125 ₽, Boosty 129 ₽).
- Лимиты на AI-вызовы, не на письмо. Paywall на аудио, обложки/иллюстрации, видео для оригиналов.
- ⚠️ «Скрытие AI-generated» **не монетизируем** (решение 2026-07-03: 86% требуют раскрытия AI, сокрытие — репутационная мина). Монетизируем качество.

## Финансовый ориентир

- **Год 1:** 1-3k активных, $1.5-12k MRR.
- **Год 2:** 5-15k активных, $10-40k MRR.
- **Год 3+:** $30-100k MRR. Нишевый бизнес для соло-фаундера, план на 2-3 года.

## Pre-MVP (параллельно с кодом)

- User research: 5-10 русскоязычных авторов (Discord фандомов, AO3-комменты, Reddit r/FanFiction, Фикбук-форумы, Tumblr). Не «вот моя идея», а «как ты пишешь сейчас, что бесит, чем пользуешься».
- TOS под tool-not-platform позицию.
- Брендинг: проверить юр. чистоту «Headcanon», застолбить домен и соцсети.

## Еженедельный self-review

Каждый понедельник 15 минут: что сделано / какая метрика двинулась / что строила НЕ туда / одна вещь с максимальным эффектом на неделю.
