## Phase 1 — MVP (1 неделя кода)

**Цель:** работающий end-to-end флоу: ввод → текст-главы → читалка → публикация.

- **День 1 — Каркас.** Next.js + Tailwind, Supabase Auth, Prisma-схема (users, stories, chapters, characters, character_references, scenes, likes, jobs).
- **День 2 — Генерация истории.** Форма «фандом, персонажи, тропы, длина» → LLM → DB. Параллельно: scene parser (разбивает главу на JSON), auto-extract character bible.
- **День 3 — Читалка.** Mobile-first компонент чтения, прогресс по тексту, переключение глав, профиль.
- **День 4 — Визуалы.** Pipeline: scene → промпт → Flux → сторадж. Семантический кеш по хешу JSON. **Async через очередь сразу** — не вешать UI на 30-сек ответ.
- **День 5 — Социалка минимум.** Хронологическая лента, лайки, сохранения, публичная страница истории с шерингом.
- **День 6 — Полировка.** Mobile UX (свайпы, шрифты, длина строки), onboarding (3 экрана), error/loading states, тёмная тема по умолчанию.
- **День 7 — Контент и боевой деплой.** 5–10 готовых историй в выбранном фандоме, end-to-end прогон, чинение находок, деплой на прод-домен.

> Платежи и подписку в первую неделю **не вкручивать**. YooKassa добавится за полдня после того как сама убедишься, что флоу нравится.

## Phase 2 — Validation (2–3 недели после MVP)

**Цель:** валидировать спрос на реальной аудитории.

- **TikTok demo run.** 5–10 коротких видео-сценок по разным тропам с фандом-хештегами. Метрики: просмотры >5k хотя бы на 2–3 видео, save rate >5%, комменты «как ты это сделала?».
- **Лендинг + waitlist** с публичным счётчиком (build-in-public маркетинг).
- **Pre-sales за $5.** Самый жёсткий тест. 50 реальных платежей = в 100 раз сильнее 5000 email-подписей. Если не продаётся за $5 — продукт не нужен в текущей формулировке.

## Phase 3 — Watch mode и видео (4–6 недель)

**Цель:** ключевой differentiator работает.

- Image-to-video через Kling / Runway
- TTS-озвучка через ElevenLabs (voice_id из character bible)
- FFmpeg склейка эпизода
- Vertical 60-sec teaser clips для TikTok-шеринга (виральный канал)
- Async-очереди с push-уведомлениями о готовности

## Phase 4 — Виральные механики

- «Что если» — branching реальностей в любой точке истории
- Remix историй (другой юзер берёт твою и меняет с любого момента)
- Reader cameos (вставить себя как side-character)
- Live polls (читатели голосуют за развитие)
- Cross-fandom mashups (генератор кринж-кроссоверов для TikTok)
- Endless mode (бесконечный сериал)

## Phase 5 — Монетизация

- Free tier (чтение опубликованных историй + 1–2 главы/день)
- Pro подписка $5–10/мес (больше генераций, видео-режим, расширенные настройки)
- Pay-per-video для разовых полноценных эпизодов
- Whale-tier для хардкорных фанатов (~$100–300/мес за персональные сериалы)

Подробная экономика и unit-costs — в [Notion](https://www.notion.so/35b2a47b0ca1812ead4ffc8d264f0caa).
