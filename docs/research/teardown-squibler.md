# Squibler — one-prompt full-book generation (Cluster A)

- **URL / дата:** squibler.io · 2026-06-27
- **Доступ:** лендинг публично; **генерация в onboarding-демо БЕЗ логина** (Generate с лендинга → сразу пишет книгу). Полный аккаунт — через Sign Up (/onboarding).
- **Скриншоты:** screens/squibler/01–03 (desktop landing + generation + mobile)
- **Токены:** tokens/squibler-landing.json · tokens/squibler-editor.json

> Разбор завершён. Squibler — **generation-first**: один промпт → целая структурированная книга
> (аутлайн + проза) агентным пайплайном с видимыми шагами. Прямой референс для нашего secondary
> generator-path, но «вся книга за раз», не «история».

### Продукт в двух абзацах
Squibler — «AI Book and Novel Writer»: опиши идею → AI генерит **полноразмерную книгу/сценарий**,
организованную в главы и сцены, затем правишь «живой документ» интеллектуальными редактор-инструментами.
Позиционирование — «от идеи до напечатанной книги быстрее, чем когда-либо», self-publishing наклон
(обложки, аудиокниги, печать, отслеживание продаж — из FAQ). Засветка: WIRED, Fast Company, Goodreads.

Аудитория шире нашей (нон-фикшн, мемуары, «заработай из дома»), но механика генерации — то, что нам
полезно для generator-path (W4-02): **минимальный фрикшн + структурированный результат + агентность напоказ**.

### Каталог фич (с лендинга)
- **Full-Length Book Generation** — идея → структурированный первый draft (главы/сцены), «живой документ».
- **Full-Length Screenplay Generation** — сценарий с форматированными сценами/диалогами.
- **Story Outline Generation** — детальный аутлайн как гибкий blueprint (до или после драфта).
- **AI Smart Writer** — co-author: анализирует текст, ставишь цель → предлагает ревизии, итерируете;
  **ты accept/reject/refine каждое предложение** (как Sudowrite Rewrite).
- **Elements Creation** — структурные персонажи/сеттинги/детали, на которые AI **ссылается при письме**
  для консистентности (= character bible / наш W2-02).
- **Visuals Generation** — выдели пассаж → генерь **картинки или короткие видео** по сцене
  (storyboard, ассеты для питча). = пересекается с нашим video-path, но для всех.

### Generation flow (главное — снято вживую)
Лендинг: **бокс генерации прямо на главной** («Let's craft your story» → **Generate →**) — инстант-вход.
Ввёл одну строку → редирект в `/onboarding/<id>` → **AI пишет книгу в реальном времени**: → `02`.
- **3 панели:** слева — авто-аутлайн (главы→сцены вложенно: «The Vigil and the Routine» → 4 сцены,
  «The Disruption» → …); центр — манускрипт (сгенерированная первая глава прозой); справа — **agentic
  чат-лог процесса**.
- **Агентность напоказ:** правый чат показывает шаги как разговор: твой промпт → «It is a pleasure…
  I will prepare a detailed outline» → шаг **«Outlining your idea»** → «I have structured your narrative
  into a compelling arc…» → шаг **«Writing first chapter»** → «opening pages finalized…» → инпут
  **«Let's finish your story»** (продолжаешь чатом). Это loading/progress как нарратив, не спиннер.
- Топ-тулбар: edit (pencil) + download/export.
- **Без логина** в демо — мощный top-of-funnel (мгновенная ценность до регистрации).

### Состояния
- **Generating:** агентные шаги-карточки в чате («Outlining…», «Writing first chapter…») + текст
  появляется в манускрипте — прозрачный прогресс.
- **Empty (left draft panel):** пустой плейсхолдер-блок пока генерится структура.
- **Mobile (375px):** манускрипт на весь экран (Montserrat-заголовок, серифная проза, комфортная
  строка), боковые панели за иконками (feather + AI-молния). Адаптивно/читаемо. → `03`.

### Дизайн-система (из токенов — полный `--sq-*`/`--color-*` набор)
- **Тёмный slate-grey:** bg `#181D20`, card `#252E32`, field `#364349`, шкала grey-000…900 до `#F4F5F5`.
  Холодный сине-серый (teal-tinted), не navy/zinc.
- **Primary — синий `#2F94C6` / `#6CB9E0`**; **gold `#FFD500`** (премиум-акцент); success `#34B273`,
  error `#E64C66`.
- **Шрифты:** дисплей — **Montserrat**; UI — **Inter**; проза — сериф (ui-serif/Georgia). Tailwind v4
  (oklab-цвета, `--text-*`/`--spacing` токены, полная radius-шкала sm→4xl).
- Темпер: «продуктовый», чистый, SaaS-полированный. Синий+золото = «софт для авторов/self-publishing».

### 🎯 Что крадём (подход, не код)
- **Инстант-генерация с лендинга/без логина → мгновенная ценность** — сильнейший top-of-funnel для
  нашего **generator-path (W4-02)** как виральный канал: ввёл идею по фандому → сразу глава.
- **Агентность напоказ** (чат-шаги «Outlining → Writing first chapter» вместо спиннера) — отличный
  паттерн loading/streaming для наших длинных генераций (W3 streaming, generator). Снимает тревогу ожидания.
- **Один промпт → структура (аутлайн) + проза одновременно** — для generator-path: не просто текст, а
  главы/сцены в навигации (ложится на нашу структуру книга→главы, W1-03).
- **AI Smart Writer accept/reject/refine** — подтверждает card/diff-модель приёма (как Sudowrite); наш W3-03.
- **Elements Creation** = ещё одно подтверждение «структурная база сущностей → консистентность» (W2-02/03).
- **Continue-by-chat** («Let's finish your story») — генерацию продолжаешь разговором; гибридная
  модель для нашего AI-чата (W3-01) + generator.

### 🚫 Чего избегаем
- **«Вся книга за один промпт»** как primary-обещание — это анти-writer-first (наш primary — автор
  пишет сам, AI ассистирует). У Squibler это годится только для нашего **secondary** generator-path.
- Self-publishing/«заработай из дома»/нон-фикшн позиционирование — не наша ниша (фанфик-комьюнити, RU).
- Синий+золото SaaS-палитра и Montserrat/Inter — не наш Editorial-Y2K (баклажан/амбер, Bodoni/Garamond).
- Широкий разброс форматов (сценарии/мемуары/нон-фикшн) — мы фокусируемся, не распыляемся.
