## Frontend

- **Next.js (App Router) + TypeScript** — подтверждено 2026-07-02 против Angular 22 / spartan-ng: LLM-ассистенты пишут React заметно лучше (Web-Bench: Claude pass@2 65% vs 40%), TipTap не имеет официальной Angular-интеграции, миграция ~11,5 тыс. строк готового кода не окупается ничем
- **Tailwind CSS v4** с подключённым `handoff/tailwind.preset.js`
- **Шрифты через `next/font`:** Bodoni Moda (display), EB Garamond (body), DM Sans (UI), JetBrains Mono (timecodes/badges)
- **Mobile-first, тёмная тема по умолчанию** (late-night reading)
- **Writer editor: TipTap** (ProseMirror, headless) с минимальным конфигом — абзацы, заголовки, bold/italic, scene-break. Хранение глав в markdown, чтобы читалка рендерила `Chapter.text` без переделки. **Решено: не Lexical** (избыточно для MVP), не голый textarea (нужен Notion-feel для авторов-нетехнарей).

Дизайн-направление — **Editorial Y2K** (микс кино-журнальной типографики и Y2K-энергии): глубокий баклажан + тёплый амбер как свет свечи + хром-голограмма на Watch CTA, малярные скотч-теги и burst-стикеры на постерах, маркизы-тикеры. Серифная типографика, drop caps в ридере, justified body.

**Источник правды по дизайну:** [`/handoff`](../handoff/README.md). Подключение токенов и пресета описано в [handoff/TASKS.md](../handoff/TASKS.md) → M0-02, M0-03. Альтернативные направления (Cozy Cinematic / Y2K Tumblr / Anime) — в [Notion](https://www.notion.so/35b2a47b0ca181b8b0bdfcda42a084b8), отброшены в пользу Editorial Y2K.

## Backend / БД

> SQLite/Pocketbase для UGC-платформы с лайками, фидами и видео-ассетами — тупик. Postgres с первого дня. Managed Supabase из РФ проблемен (Stripe-биллинг, периодические блокировки), поэтому self-host — но на российском VPS, не на Hetzner (см. Деплой).

| Вариант                                                      | Когда использовать                                                                                                                                                                                                                                                                                                                                  |
| ------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Self-hosted Supabase на RU-VPS (Timeweb / Selectel)** ⭐   | MVP-выбор. Postgres + Auth (GoTrue) + Storage + Realtime + автогенерируемый REST/GraphQL в одной развёртке. Coolify-шаблон поднимается за час, но ⚠️ известные грабли (офиц. тред Supabase, 2025–2026): обновления без changelog, устаревшие версии сервисов в шаблоне, Logflare ест ресурсы — закладывать время на настройку и осторожные апдейты. |
| Managed Postgres (Timeweb / Selectel / Yandex) + Better Auth | Вариант упрощения для прода: self-hosted Supabase — самая хрупкая часть стека, а Better Auth в 2026 — консенсусный дефолт для новых Next.js-проектов. Цена — переписывание auth-кода; решение отложено до подготовки прода.                                                                                                                         |
| Managed Supabase (EU)                                        | Запасной через посредника для оплаты. Дёшево, но риск отвала биллинга. Сейчас используется для dev (Supabase Cloud).                                                                                                                                                                                                                                |

## ORM

| Вариант       | Заметки                                                                                                                                                                                                                                                                  |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Prisma** ⭐ | Схема в одном файле, миграции человеческие, отличный DX. Работает поверх Supabase Postgres напрямую (Supabase JS-клиент использовать необязательно). Prisma 7 (ноябрь 2025) удалил Rust-engine: запросы до 3.4x быстрее, бандл −90% — аргумент «Prisma тяжёлый» устарел. |

## Деплой

- **Coolify** на собственный VPS — self-hosted Vercel-аналог. Бесплатно, ставится за час, поддерживает Next.js, авто-деплой из Git.
- **VPS: Timeweb Cloud или Selectel (РФ)**, с запасом RAM под Supabase (4GB+). Оплата рублями, managed Postgres и S3 рядом, у Timeweb гранты до 1 млн ₽. RU-VPS также закрывает 152-ФЗ (первичная запись ПДн в РФ; штрафы с 2026 до 18 млн ₽ / 3% выручки) и доступность сайта для модерации и вебхуков YooKassa.
- ❌ **Не Hetzner / OVH / DigitalOcean** — с июня 2025 РКН через ТСПУ режет HTTPS к их диапазонам (обрыв после ~16 КБ, жёстче всего в мобильных сетях); на середину 2026 не разблокировано. Cloudflare как прикрытие не работает — сам под блокировкой. Бонус-проблемы Hetzner: прекратил обслуживание клиентов из РФ, оплата только через посредников.
- ❌ **Не Vercel/Netlify** — проблемы оплаты, бан-волны на РФ-IP, плюс тоже попадают под ТСПУ-блокировки.
- ⚠️ **SSE-стриминг за Traefik (Coolify):** известные грабли с буферизацией — заголовок `X-Accel-Buffering: no`, отключить compress-middleware на стрим-роутах, возвращать `ReadableStream` сразу. Обязательно e2e-проверять стрим через прод-прокси, не только локально.
- Next.js self-hosted — официально first-class путь (гайд обновлён 03.2026): `output: 'standalone'` + Docker + `next start`; ISR и image optimization на одном инстансе работают из коробки, OpenNext не нужен. Для джобов >5 сек — worker-процесс (BullMQ) на том же VPS, отдельный API-фреймворк не заводим.

## Tier-структура (Phase 5, закладывается заранее)

| Tier       | Цена       | Что входит                                                                                                                                                   |
| ---------- | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **FREE**   | $0         | Чтение всех публичных историй; письмо без лимитов; 3-5 AI-генераций/мес; AI-помощник writer-mode (лимит запросов); свои AI-generated видны всем по умолчанию |
| **PRO**    | $10-15/мес | 30-50 AI-генераций; аудио-прослушивание; скрытие своих AI-generated из публичной ленты; иллюстрации к сценам; без рекламы                                    |
| **STUDIO** | $25-40/мес | Всё из PRO + видео для оригинальных историй (moments, лимит); расширенные world-building tools; аналитика по своим историям; приоритетная очередь            |
| **WHALE**  | $100+/мес  | (позже, по запросу) Эксклюзивные полнометражные сцены для оригинальных миров; personal consulting                                                            |

Воронка: free → PRO (читатели/казуальные) → STUDIO (серьёзные авторы) → WHALE. Аналог Suno, ElevenLabs, NovelAI. Лимиты на AI-вызовы, не на письмо в черновике.

## Архитектурный совет

**Устарело (2026-07).** Аргумент «EU-сервер ради прямого AI-доступа» больше не работает: OpenRouter и Anthropic режут по биллинг-профилю аккаунта, а не по IP сервера. Актуальная раскладка: **всё на RU-VPS (app + Postgres + storage), AI-вызовы через российские агрегаторы** — они одинаково работают с RU- и EU-сервера. EU-нода понадобится только под глобальный запуск или зарубежный GPU/сторедж — и тогда как origin за российским фронтом, не как основной сервер.
