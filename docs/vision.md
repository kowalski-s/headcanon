# Vision

> Инструмент для авторов фанфиков с AI-помощником + two-sided market. Автор пишет историю сам, AI помогает (редактор, консультант, world-building); читатели читают и слушают; «ленивые создатели» получают сгенерированную историю как виральный канал. NovelAI × Sudowrite × AO3, заточенные под фандомное комьюнити.

> Полное обоснование пивота с v1 — [Notion: 🔄 Pivot v2: Two-sided model](https://www.notion.so/36b2a47b0ca181669ea3ce37d9893091).

## Идея в одном абзаце

Человек пишет фанфик сам — главу за главой, в приватном черновике, сколько угодно. AI работает рядом как помощник: задаёт вопросы для движения сюжета, напоминает о деталях мира, редактирует, помогает с world-building, достаёт персонажей из текста в character bible. Готовую историю можно опубликовать, дать прочитать или прослушать (аудио). Для тех, кто писать не хочет, есть вторичный путь — генерация истории по промпту. Видео — только для оригинальных миров автора, отложено.

## Два пользовательских пути

- **Writer path (PRIMARY).** Человек сам пишет, AI ассистирует. Это ядро продукта и основной источник качественного контента, который держит читателей.
- **Generator path (SECONDARY).** Человек получает сгенерированную историю по промпту. Виральный top-of-funnel канал, не основная аудитория. AI-generated публикуются по умолчанию в отдельной категории.

## Целевая аудитория

- **Primary — real authors с AI-помощником.** Уже инвестируют время в писательство, готовы платить за инструменты ($15-40/мес), создают контент, лояльны (инструмент = часть процесса), несут меньший юридический риск (сами пишут, сами отвечают). К ним применима whale-стратегия.
- **Secondary — lazy authors (генеративные).** Виральный канал привлечения. Низкое удержание, высокая шеримость. Конвертируются в writers и readers со временем.
- **Tertiary — чистые читатели.** Приходят на UGC от real authors. Монетизируются подпиской (аудио, безлимит чтения). Создают engagement-сигналы для топов.

Late-night reading (60%+ потребления между 22:00-02:00), mobile-first, тёмная тема по умолчанию.

## Первая ниша и география

- **География: РФ-first.** YooKassa уже выбрана, инфраструктура знакома. Потолок ~$50-100k/год. Архитектура закладывается так, чтобы не запереть глобал позже.
- **Первая ниша: русскоязычные авторы (Фикбук-сегмент).** Конкретное комьюнити с конкретными людьми, доступное основателю как инсайдеру. Beachhead-фандомы: HP, All for the Game, Naruto, JJK.

## Core insight

- **Юридическая защита масштабируется с размером.** Маленький продукт — DMCA safe harbor достаточен. Видео каноничных персонажей — красная зона, остальное (текст, аудио, иллюстрации) живёт. Подробнее — [architecture.md → Юридический инвариант](architecture.md).
- **Tool, not platform.** Юзер сам отвечает за контент (TOS), платформа не имеет ownership. Классическая стратегия YouTube/GitHub/Wattpad.
- **Качество vs масштаб.** Real authors создают контент, который держит читателей. Чистая генерация надоедает через 2-3 главы.
- **Экономика текста и аудио сходится.** Текст ~$0.05/глава; TTS ~$0.30/мин, глава 5-10 мин = $1.5-3 при подписке PRO $10-15/мес. Видео ($1-3/5-сек клип, $25-40/эпизод) — премиум, отложено, дешевеет ~5x/год.

## Стратегические якоря

- **Writer-first.** Primary-ценность — инструмент для письма, не генерация. Generation — secondary.
- **Tag-based video gating с первого дня.** Каноничный фандом в тегах → видео отключено в коде. Не «приписать потом».
- **Mobile-web первые 6+ месяцев**, не нативное приложение.
- **Free tier для чтения и письма обязателен** — без него нет вирала. Платное — AI-генерации сверх лимита, аудио, скрытие AI-generated, видео для оригиналов.
- **Адаптация под российскую инфраструктуру** — платежи (YooKassa), хостинг (Hetzner DE), AI-доступ (OpenRouter / прокси).
- **Это нишевый бизнес для соло-фаундера, не единорог.** План на 2-3 года. Год 1: $1.5-12k MRR. Год 2: $10-40k MRR.
