# Vision

> AI-платформа для генерации фанфиков с возможностью смотреть свои истории как мини-сериалы. Wattpad × DramaWave × AI Dungeon, заточенная под фандомное комьюнити.

## Идея в одном абзаце
Юзер выбирает любой фандом, персонажей, тропы — ИИ генерирует фанфик в формате глав. Текст можно читать как обычно или переключиться в **Watch mode**, где глава автоматически проигрывается видео-клипами с TTS-озвучкой. Истории публикуются, шерятся, ремиксятся другими пользователями.

## Целевая аудитория
- **Ядро:** фанатичные читатели/писатели фанфиков 16–30 лет
- **Late-night reading:** 60%+ потребления Wattpad между 22:00 и 02:00
- **Mobile-first:** вертикальные форматы, читалка под телефон
- **Фандомы:** универсальная поддержка с первого дня. Юзер вводит любой фандом и персонажей текстом, LLM использует знания из претрейна. Никакой ручной курации канонов командой.

## Core insight
- Текст-генерация ~$0.05 за главу — массовый бесплатный контент
- Видео-генерация ~$25–40 за эпизод — премиум для ключевых сцен
- **Wattpad-модель:** 1% генерит, 99% потребляет — стоимость видео амортизируется через просмотры
- Видео дешевеет ~5x в год → продукт становится экономически выгоднее со временем

## Стратегические якоря
- **Универсальный продукт + сфокусированный маркетинг.** Продукт работает с любым фандомом с первого дня. Маркетинг бьёт в одно конкретное комьюнити (TikTok, треды, мемы) для построения первого ядра — это messaging-стратегия, не продуктовое ограничение.
- **Mobile-web первые 6+ месяцев**, не нативное приложение.
- **Free tier для чтения обязателен** — без бесплатного чтения нет вирала.
- **Адаптация под российскую инфраструктуру** — платежи, хостинг, AI-доступ.
