# Handoff: headcanon — writer-first pivot v2 (полный набор экранов)

## Overview

**headcanon** — платформа фанфикшена с ИИ-соавтором. Направление: **writer-first pivot v2** —
primary-поверхность это инструмент писателя (тихий редактор + соавтор-чат + библия персонажей +
граф связей), лента/чтение/генерация — secondary.

Эстетика: **Editorial Y2K** — «тихий письменный стол в шумной редакции глянцевого литжурнала.
2 часа ночи, свеча горит». Тёмный баклажан вместо нейтрального чёрного, амбер + роза, Bodoni-курсив
как голос продукта.

Этот пакет покрывает полный путь пользователя: лендинг → регистрация/первый вход → главная →
мой стол → редактор (3 режима) → AI-соавтор (чат) → граф связей → библия → кабинеты → онбординг.

## About the Design Files

Файлы в `prototypes/` — **дизайн-референсы, созданные в HTML/React (inline-стили, Babel standalone)**.
Это прототипы, показывающие точный вид и поведение, а НЕ production-код для копирования.
Задача — **воссоздать эти экраны в среде целевой кодбазы** (Next.js/React/Vue/что есть),
её паттернами и библиотеками. Если кодбазы ещё нет — выбери подходящий фреймворк сам
(дизайны написаны как React-компоненты, поэтому React/Next.js — наименьшее трение).

Открыть прототипы: открой `prototypes/headcanon.html` в браузере (нужен интернет для React CDN
и Google Fonts). Все экраны лежат на одной пан/зум-канве, секции 00–09.

**Два обязательных документа рядом:**
- `DESIGN.md` — канон визуальной ДНК: токены, типографика, запреты (AI-slop и пр.)
- `DESIGN-writer.md` — арт-дирекшн writer-поверхностей: принцип «тихий центр, выразительные края»,
  тон AI, запреты v2. При конфликте по токенам главнее DESIGN.md, по продукту — DESIGN-writer.md.

## Fidelity

**High-fidelity.** Цвета, типографика, отступы, копирайтинг и интеракции — финальные.
Воссоздавать пиксель-точно, но компонентами и паттернами целевой кодбазы.

## Screens / Views (canvas-секции 00–09)

### 00 · Путь пользователя
- **Лендинг** (desktop `PvLandingDesktop` + mobile `PvLandingMobile`, `screens-pivot9.jsx`) —
  hero «Полночное чтиво, которое пишешь ты», веер из 3 обложек 2:3 (наклон ±7°, свечной glow),
  CTA «начать писать — бесплатно» + secondary «☾ просто читать», 3 фичи-пропа,
  бейдж честности «человек · AI-ассистировано».
- **Первый вход после реги** (desktop `PvFirstRunDesktop`) — guided start: шаг 1 выбор пути
  (3 карточки: пишу сама ★ / сгенерируй мне / просто читать), шаг 2 фандом-чипы,
  CTA «за стол →», «пропустить — осмотрюсь сама». На мобиле роль этого экрана играют
  онбординг-развилка + «пустой стол» (секция 09/05).
- **Главная залогинена** (desktop `PvHomeDesktop`) — writer-блок с momentum
  («Драко всё ещё у окна, гл. 7 ждёт» + продолжить) primary, generator-блок secondary,
  лента «этой ночью читают» (5 обложек с тегами «новая гл.», «ты на гл. 3», «твоя»).
  Mobile-версия — `PvHomeMobile` в `screens-pivot2.jsx` (2 блока + тизер ленты + таб-бар).

### 05 · Мой стол (дашборд писателя) — `screens-pivot6.jsx`
`PvDeskDesktop`, `PvDeskMobile`, `PvDeskEmptyMobile`.
- Momentum-подводка литературным голосом («Три ночи назад ты оставила Драко у окна»), CTA продолжить.
- Полка обложек 2:3: с градиент-обложкой ИЛИ **типографская обложка** для черновиков без арта
  (Bodoni-титул на surface, амбер-хэйрлайн, mono-статус `ЧЕРНОВИК · ГЛ. 7 · 12 480 СЛ`);
  пустой черновик — пунктирная рамка.
- Momentum-панель справа: стрик «9 ночей подряд», спарклайн слов за 14 ночей, прогресс серии.
- Пустой стол = guided start (фандом-чипы + один CTA), НЕ blank canvas.

### 06 · Редактор — тихий центр — `screens-pivot5.jsx`
`PvQuietEditorDesktop`, `PvQuietEditorMobile`, проп `light` (тема «бумага»), живой стейт `mode`.
- **Ragged-right** (textAlign left) при письме, колонка max 620px, Garamond 17–18px/1.62.
- Chrome минимальный, тает до opacity .16 в фокус-режимах.
- 3 режима (переключатель внизу): **письмо** (ghost-подсказка с TAB, slash-строка на мобиле) /
  **фокус** (не-активные абзацы opacity .3 + радиальное затемнение) /
  **печ. машинка** (активный абзац строго по центру вертикали: flex-колонка
  `1 1 0 / active / 1 1 0`).
- Автосейв — mono-ярлык «сохранено» с амбер-точкой. Никаких тостов.
- ЗАПРЕЩЕНО: зерно/стикеры/glow поверх текста, drop cap при письме.

### 07 · AI-соавтор — чат о тексте — `screens-pivot8.jsx`
`PvAssistDesktop`, `PvAssistMobile`. Чат-первый:
- Правая панель 340px (mobile: bottom-sheet на весь экран) — **живой чат**: соавтор замечает
  противоречия по главам (с чипами «да, исправь / так задумано»), отвечает на свободные вопросы
  про рукопись, даёт 3 варианта хода (карточки с ♡ «оставить» и «→ в текст»).
- Инпут «спроси что угодно о своём тексте…», Enter отправляет; ожидание —
  «перечитываю твою сцену…» с дышащей амбер-точкой (НЕ спиннер).
- В центре — inline-механика: slash-затравка → «развернуть» → прогресс-повествование
  (3 строки Bodoni-курсивом проявляются) → **amber-pending вставка** (рамка amber66,
  левый борт 2px amber, текст курсивом, стриминг по символам) → карточка
  «в текст ✓ / иначе ↻ / мимо» + подпись «человек · AI-ассистировано».
- Квота «2 / 5 разворотов сегодня» — единственная прозрачность модели. Никаких температур/моделей.

### 08 · Граф связей (сигнатурная фича) — `screens-pivot7.jsx`
`PvGraphDesktop`, `PvGraphMobile`, общее ядро `PvGraphCore` (данные `PV_NODES`/`PV_LINKS`,
позиции: desktop 940×600, mobile 320×540).
- Узлы-персонажи: круглые «портреты» (градиенты обложек + зерно), размер по роли (r 26–44),
  за каждым — **дышащий свечной glow** (`pvBreathe` 3–5s, mixBlendMode screen).
- Связи — SVG-линии: **rose** = чувство/тайна, **amber** = сюжетная связь; пунктир = прошлое/слабое.
- **Клик по узлу**: его связи ярче + подписи связей Bodoni-курсивом (с обводкой цветом фона),
  несвязанные узлы/линии гаснут до opacity .25–.3. Клик по холсту — сброс.
- Легенда внизу + переключатель «граф / карточки».

### 09 · Библия · кабинеты · онбординг
- `PvBibleDesktop` (`screens-pivot2.jsx`) — карточки персонажей + правая колонка
  «✦ найдено в тексте»: AI-извлечённые факты с «в библию ✓ / мимо».
- Кабинеты (`screens-pivot3.jsx`, `screens-pivot4.jsx`) — переключатель ролей
  «читатель / автор» (амбер-пилюля). Автор: спарклайн слов, дочитывания/сохранения/подписчики,
  список историй, награды. Читатель: стрик «12-я ночь подряд, свеча не гаснет», фан-статы
  («1 284 600 слов ≈ 2,5 „Войны и мира"»), продолжить чтение с прогрессом, доли фандомов,
  мостик «✎ написать свою».
- `PvOnboardingMobile` — развилка 3 путей (mobile-аналог первого входа).

## Interactions & Behavior

- **Редактор**: переключение режимов — плавные transition opacity .45–.5s; курсор —
  мигающий амбер-каретка (`steps(1)` 1.1s).
- **Соавтор-чат**: отправка → 1.5s «перечитываю…» → ответ; автоскролл к низу.
- **Inline-разворот**: thinking 3×820ms → result, стриминг ~20ms/2 символа;
  accept вставляет текст как обычный абзац, reject возвращает slash, refine регенерирует вариант.
- **Граф**: transition opacity .4s на выделение; glow-анимация бесконечная (декоративная,
  вне контента — допустимо), уважать `prefers-reduced-motion`.
- Hover-состояния в прототипах не проработаны — использовать паттерны кодбазы
  (рекомендация: лёгкое поднятие яркости амбера, НЕ scale/тени).

## State Management

- Редактор: `mode: 'write' | 'focus' | 'typewriter'`; тема `light/dark`; активный абзац.
- Соавтор: `messages[]`, `draft`, `busy`; inline: `phase: beat→thinking→result→accepted`, `variant`.
- Граф: `selected: nodeId | null`; производное множество связанных узлов.
- Кабинет: роль `reader | author`.

## Assets

Изображений нет — обложки это CSS-градиенты (`mix-poster-grad-1…5` в `screens-mix.jsx`)
+ зерно (`mix-grain`, inline SVG-noise) + виньетка. В продакшене заменяются на
AI-обложки/арты; типографская обложка остаётся как фолбэк для черновиков.
Шрифты: Google Fonts (Bodoni Moda, EB Garamond, DM Sans, JetBrains Mono, Cormorant Garamond, Lora).
