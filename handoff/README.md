# headcanon — handoff package

> AI-генератор фанфиков по фандомам. Mobile-first, late-night reading.
> Hook: «Ship → готовая глава за <60 сек».

Этот пакет — **источник правды** по дизайну для агента, который пишет код в репозитории.
Если что-то противоречит — дизайн-система (`DESIGN.md` + `tokens.*`) важнее, чем мой случайный markup в прототипе.

---

## С чего начать (порядок чтения для агента)

1. **`DESIGN.md`** — что за продукт, принципы, дизайн-система, правила.
2. **`typography.md`** — шрифты, шкала, drop caps, локализация.
3. **`tokens.json`** + **`tokens.css`** + **`tailwind.preset.js`** — токены. Подключи preset в `tailwind.config.ts`.
4. **`screens/*.md`** — спека каждого экрана: интент, состояния, edge-cases, что передаётся с бэка.
5. **`TASKS.md`** — ordered backlog: с чего начать, что в каком спринте.
6. **`prototypes/`** — рабочий HTML/JSX-прототип. Открыть `prototypes/headcanon.html` в браузере. **Это референс, не код для копирования** — там React 18 без сборки + inline-стили; в репо всё переписывать на Next.js + Tailwind.

## Решённые продуктовые принципы

1. **Mobile-first.** Все основные пользовательские задачи решаются с телефона одной рукой в кровати, под одеялом, в 1:42 ночи.
2. **Ship — главный хук.** Создание истории всегда начинается с фандома → пейринга. Тропы, сеттинг, тон — после.
3. **Glanceable Watch mode.** Альтернатива длинному чтению: 4 коротких эпизода × ~4 минуты с озвучкой и атмосферой. Это **не** видеоплеер с лицами — это аудио + кинематографический still + субтитры.
4. **Late-night режим.** UI не светит белым в темноте. Базовая тема — dark, амбер-акценты как свет свечи.
5. **Магазинная типографика, без AI-slop.** Bodoni × Garamond, drop caps в ридере, нет «градиентных кнопок и эмоджи».
6. **Без хот-пинка.** Розовый — точечный вторичный акцент (счётчики, badges), не primary.

## Что в пакете НЕТ (и должно появиться при разработке)

- Реальные обложки историй — генерятся через Replicate/fal.ai по промпту из `{фандом, ship, тропы}`. В прототипе — градиенты-плейсхолдеры.
- Озвучка эпизодов Watch mode — ElevenLabs, голос подбирается под персонажа.
- Текст глав — стримится с OpenRouter. В прототипе захардкожено.
- Авторизация / профили — Supabase Auth.
- Платёжный wall — YooKassa, после N бесплатных глав.

## Контакт

Если что-то непонятно по дизайну — пишите в чат с дизайн-агентом, не угадывайте.
Прототип в `prototypes/headcanon.html` всегда открывается в браузере — есть Tweaks-панель справа, можно переключать тему/акцент/плотность.
