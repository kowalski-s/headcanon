# Дизайн-система headcanon

Направление: **Editorial Y2K** (микс кино-журнальной типографики и tumblr-энергии).
Не Penguin, не Tumblr 2014 — а их встреча в полночь.

---

## Визуальная ДНК

**База — кинематограф.** Глубокий баклажан, тёплый амбер как свет свечи, плёночное зерно, виньетка по краям обложек, drop caps в ридере, justified body. Кадры дышат пустотой.

**Энергия — Y2K журнал.** Маркизы-тикеры с эмодзи-звездой, sticker-всплески «★ хит» на главной обложке (повёрнуты на ±8°), малярный скотч-теги на постерах в ленте, ironic-подпись Bodoni-italic «★ полночное чтиво для тех, кто не спит ★», лёгкая голограмма (хром-градиент `#C9D4E8 → #E5A95A → #D67890`) на CTA `▸ watch`.

**Что НЕЛЬЗЯ:**
- Розовый как primary (только точечный акцент).
- Эмодзи в UI (✦ ✧ ★ ☆ — это типографские символы, можно).
- Аккуратные «дизайн-системные» rounded-corner-badge с левой цветной полосой.
- Inter / Roboto / системные санс-серифы.
- Градиентные кнопки.
- Скевоморфные иконки.
- Drop shadow на UI (тени — только на постерах: ambient depth, не lift).

## Цвета

См. `tokens.json` → `colors`. Базовые роли:

| Роль | Hex | Где |
|---|---|---|
| `bg` | `#160B22` | Полотно ленты, ридера, story page |
| `bg.alt` | `#1F1230` | Watch-mode (на тон темнее), pressed states |
| `surface` | `rgba(245,239,224,.04)` | Карточки эпизодов, NEXT-блок в ридере |
| `ink` | `#F5EFE0` | Body-копия, заголовки |
| `ink.dim` | `#A89BB5` | Метаданные (timecodes, счётчики), placeholders |
| `ink.faint` | `#6E6478` | Disabled, дальние мета |
| `amber` | `#E5A95A` | Primary CTA, drop caps, акцент-курсив, амбер glow |
| `rose` | `#D67890` | Вторичный акцент (счётчик ♡, отдельные badges) |
| `chrome.1/2/3` | `#C9D4E8 / #E5A95A / #D67890` | Голограмма-градиент на watch CTA |

## Типографика

См. `typography.md` — детали и шкалу. Стек:

- **Display:** `"Bodoni Moda", "Bodoni 72", serif` — заголовки, чаптер-номера, drop cap. Контрастный didone, **используем оба stroke и italic**.
- **Body:** `"EB Garamond", Georgia, serif` — сплошной текст глав, длинные подписи, justified.
- **UI:** `"DM Sans", system-ui, sans-serif` — кнопки, табы, формы, мета.
- **Mono:** `"JetBrains Mono", monospace` — таймкоды, EP 02 / 04, str 3 / 9, проценты прогресса. Всегда `letter-spacing: 0.18em` `text-transform: uppercase` для ярлыков.

## Motion

| Случай | Длительность / easing |
|---|---|
| Tap → переход на story page | 320 ms / `cubic-bezier(.2,.8,.2,1)` (cinematic) |
| Маркиза-тикер сверху | linear infinite, 35s/loop |
| Свечной glow «дыхание» | 6s ease-in-out, opacity 0.5 ↔ 0.7 |
| Hover на постерах (desktop) | 200 ms, scale(1.02) + tilt(-1deg) |
| Page-reveal в ридере | fade-in 240ms, **без слайда** — глаз должен «приехать» к тексту, не страница |
| Watch mode заходит | crossfade 600ms из обложки в фрейм |

**Reduced motion:** маркизу останавливаем, glow заменяем статичным радиальным градиентом.

## Иконография

- **Никаких lucide / heroicons «по умолчанию».** Всё, что в прототипе сейчас идёт типографскими символами (`☆ ✎ ♡ ◯ ← →`), оставляем как есть в табе и нав-баре.
- Если **нужна** настоящая иконка (например, share-action) — рисуем custom-stroke в `1.5px` амбер-stroke, иначе не вписывается. Никаких заливок.

## Состояния и интеракции, которых нет в прототипе

Прототип — статичный «момент времени». В коде надо реализовать:

- **Skeleton-состояния** для постеров и эпизодов (амбер-shimmer на `surface`).
- **Empty states** для ленты (только что зарегистрированный пользователь): крупный Bodoni «выбери первый фандом → ст», CTA `+ начать` с амбер-glow.
- **Error states** при сбое генерации главы: Bodoni-italic «глава застряла в подземельях», retry-pill.
- **Loading генерация главы** (стриминг): по символам в ридер, drop cap появляется первой, дальше ткёт текст с амбер-курсором.
- **Pull-to-refresh** в ленте: тащим вниз → проявляется маленький свечной круг, отпускаем → один новый постер въезжает сверху с амбер-flash.
- **Long-press на постере** в ленте: всплывающий бэкдроп-блюр + 3 действия (сохранить / поделиться / в Watch).

## Локализация

UI — русский. Контент историй — может быть смесью ru + транслит-имён фандома.

Bodoni Moda и EB Garamond поддерживают кириллицу (важно — не EB Garamond `Latin Extended` only, а полная версия с Cyrillic subset из Google Fonts).

Все строки — через `next-intl` или эквивалент, в коде ничего не хардкодим. Готовый словарь в `i18n/ru.json` → потом легко добавить английский.

## Доступность

- Контраст: `ink (#F5EFE0)` на `bg (#160B22)` = ~16:1 ✓. Body всегда на `ink`. `ink.dim` (#A89BB5 на bg) = ~6.4:1 — ОК для мета и подписей, **не для основной копии**.
- Минимальный hit target — 44px (iOS HIG). Pill-CTA в Create — 48px высоты. Tab bar — 56px.
- `prefers-reduced-motion`: см. секцию Motion.
- Скрин-ридер: типографские символы (`☆ ♡ →`) маркируем `aria-hidden`, рядом — текстовый лейбл.
- Drop cap должен читаться скрин-ридером как часть слова, не отдельным символом — оборачиваем `<span aria-hidden>С</span>` + остальное в `<span>тегда снег падал…</span>`.

## DO / DON'T (ускоренная сводка для код-агента)

✅ Bodoni для заголовков, Garamond для body, DM Sans только для UI-чрома.
✅ Амбер на чёрно-баклажане. Розовый — точечно.
✅ Курсив — это не «акцент», это голос продукта. Используй смело.
✅ `text-wrap: balance` на заголовках, `text-wrap: pretty` на body.
✅ Drop cap через `float: left` + `font-feature-settings`. Не через `::first-letter` — слишком хрупко.

❌ Не добавлять иконки lucide рядом с типографскими символами.
❌ Не делать «cards-everywhere»: в ленте постер живёт сам по себе, без карточки-обёртки.
❌ Не использовать `bg-gradient-to-r from-amber-400 to-rose-400` на CTA — это AI-slop. Голограмма — только на Watch-CTA, как фирменный знак.
❌ Не центровать длинный body текст. Justified или ragged-right.
❌ Не вставлять backdrop-blur на полстраницы — только на sticky-CTA внизу.
